Question Areas to Prepare

  • Core Java: collections, exceptions, streams, concurrency, and OOP tradeoffs
  • Spring Boot: auto-configuration, dependency injection, controllers, services, repositories
  • REST APIs: status codes, validation, error handling, pagination, idempotency
  • Data: SQL joins, indexes, transactions, JPA mappings, N+1 query problems
  • Testing and delivery: JUnit, Mockito, Docker, CI/CD, logs, and production debugging
