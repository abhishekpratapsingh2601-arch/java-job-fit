JavaJobFit

Privacy Policy

Effective date: June 29, 2026

JavaJobFit processes raw resume text and raw job description text to generate an ATS-style scan. Raw resume text and raw job description text are not stored.

Data Collected

JavaJobFit may store generated report output, ATS-style scores, matched strengths, missing keywords, top fixes, resume bullet suggestions, interview questions, prep-plan content, experience level, report ID or public ID, timestamps, optional email, optional country, optional feedback, consent status, source, and simple event metadata.

Data Not Collected

JavaJobFit does not store raw resume text, raw job description text, or uploaded resume files. Upload support is not live in this MVP.

Generated Reports

Generated report output may be stored so the app can display scan results, validate beta flows, and improve JavaJobFit. Generated output should not be treated as a hiring guarantee.

Email, Feedback, And Events

If you submit an email, feedback, or a beta event, JavaJobFit may store that optional information. Feedback should not include raw resume text, raw job descriptions, passwords, secrets, or highly sensitive personal data.

Deletion Requests

Users may request deletion of optional email, feedback, report data, or related metadata by contacting JavaJobFit. This is intended to be simple and friendly for India DPDP and GDPR-style privacy expectations, without pretending this solo beta tool is a large legal department.

Use Carefully

Do not paste highly sensitive personal data that you do not want processed, such as government ID numbers, financial information, medical information, passwords, or secrets.
